# lotus
Small Minecraft Texture Pack with Shader Support

## requirements
[node.js & npm](https://nodejs.org)

## building
1. clone with git(hub)
2. `npm install`
3. `gulp zip-copy` (copies to %appdata%) or `gulp zip` (copies to ./build)
